Visão Geral do Tema
Arquitetura e tecnologias utilizadas

O tema Clínica Nodari é um tema WordPress customizado, construído do zero. Ele não usa Elementor, Divi, nem nenhum page builder — todo o layout é controlado por arquivos PHP e CSS próprios, o que garante máxima performance.

🔧 Tecnologias

WordPress + ACF (Advanced Custom Fields) + Yoast SEO + PHP + CSS puro

🎨 Identidade Visual

Fonte Century Gothic (self-hosted), paleta Teal + Amarelo conforme manual TAI Branding 2025

⚡ Performance

JS com defer, imagens com loading=lazy, fontes pré-carregadas, sem dependência de CDN externo

📱 Responsivo

Breakpoint em 960px. Menu hamburguer automático no mobile, layout em coluna única

O Hero é a seção de topo da página inicial, com título, descrição, botões de ação e imagem lateral. Todos os textos e a imagem são editáveis pelo ACF.

Foto do Psicólogo — Guia Completo
Como adicionar e substituir a foto de cada profissional
Abra o post do psicólogo WP Admin → Psicólogos → clique no nome da profissional para editar
Localize o box "Psicólogo" na sidebar direita Role a página para baixo. Na coluna da direita (ao lado do editor), procure o box chamado "Psicólogo" ou "👤 Foto do Psicólogo".
Clique em "Adicionar imagem" Abrirá a biblioteca de mídia do WordPress. Selecione ou faça upload da foto.
Confirme a seleção A miniatura da foto aparece no box. Clique em "Atualizar" para salvar.
📐

Especificações ideais da foto: 400×500 pixels (orientação retrato), fundo neutro ou desfocado, rosto centralizado, formato JPG ou WebP. Tamanho máximo: 1MB.

Como o tema prioriza a foto

O tema segue esta ordem de prioridade ao exibir a foto:

  1. 1º — Campo ACF "Foto" (box Psicólogo na sidebar) ← recomendado
  2. 2º — Imagem Destacada do post (caso o campo ACF esteja vazio)
  3. 3º — Nenhuma foto é exibida (o card não quebra, apenas omite a imagem)
⚠️

Se o box do ACF não aparecer na sidebar, vá em Tela → Opções de Tela (canto superior direito) e verifique se "Psicólogo" está marcado para exibição.

Área de Comentários
Como funciona, moderar e configurar

Os comentários aparecem automaticamente ao final de cada post do blog, desde que estejam habilitados nas configurações do WordPress.

Layout do formulário

O formulário tem:

  • Nome e E-mail lado a lado (obrigatórios)
  • Textarea de comentário em largura total
  • Botão "Comentar" em teal arredondado
  • Campo "Site" e "Salvar dados" ocultados por design

WP Admin › Comentários

Por padrão o WordPress requer aprovação manual. Para aprovar: passe o mouse sobre o comentário → clique em "Aprovar". Comentários aprovados aparecem no post.

SEO em Posts do Blog
Configuração ideal para cada artigo

1. Frase-chave de foco

É a palavra ou expressão que o Google deve associar ao post. Use algo que o leitor digitaria para encontrar o artigo.

  • ✅ como funciona a terapia TCC
  • ✅ ansiedade tratamento psicológico
  • ❌ Clínica Nodari blog post (muito genérico)

A frase-chave precisa aparecer literalmente no corpo do texto para o Yoast ficar verde.

2. Título SEO

É o título que aparece nos resultados do Google (diferente do título do post). Use as variáveis clicando em "Inserir variável":

%%title%% | %%sitename%%

Limite: entre 50–60 caracteres. A barra fica verde quando está nessa faixa.

Barra laranja = muito curto ou muito longo

Barra verde = tamanho ideal

3. Slug

A URL do post. Regras obrigatórias:

  • Somente letras minúsculas
  • Sem acentos (ã → a, ç → c, ê → e)
  • Hífens no lugar de espaços
  • Inclua a frase-chave principal
❌ Ruim:  Terapia Cognitivo-Comportamental o que é
✅ Bom:   o-que-e-terapia-cognitivo-comportamental

4. Metadescrição

Texto que aparece abaixo do título nos resultados do Google. Limite: 120–155 caracteres.

  • Inclua a frase-chave de foco
  • Termine com ponto ou exclamação
  • Seja direta e convide ao clique
Entenda como funciona a Terapia Cognitivo-Comportamental (TCC)
e como ela pode ajudar a tratar ansiedade e depressão.
Agende sua consulta na Clínica Nodari!

Variáveis de Cores e Espaçamento
Como mudar a identidade visual sem quebrar o tema

Todas as cores do tema são definidas em variáveis CSS no topo do main.css. Para mudar uma cor em todo o site basta alterar o valor da variável:

/* Paleta principal — Manual TAI Branding 2025 */
:root {
  --color-teal:        #00a6af;  ← cor principal
  --color-teal-deep:   #0e6e73;  ← hover e destaque
  --color-teal-light:  #4dc8cf;  ← versão clara
  --color-teal-bg:     #e3f5f6;  ← fundos suaves
  --color-yellow:      #fcc81f;  ← botões e destaques
  --color-dark:        #1a2b3c;  ← textos escuros
  --color-text:        #2D3748;  ← corpo de texto
  --color-text-muted:  #718096;  ← texto secundário
}
💡

Para mudar a cor principal do site de teal para outra cor, basta alterar --color-teal. Todos os botões, links e destaques mudam automaticamente.

Comportamentos JavaScript
Animações e interações automáticas do site

O arquivo assets/js/main.js controla três comportamentos automáticos:

ComportamentoComo funcionaComo usar
Menu hamburguer No mobile, o botão ☰ abre/fecha o menu. Fecha automaticamente ao clicar em qualquer link ou fora do menu. Automático
Nav sticky com sombra Quando o usuário desce a página mais de 40px, a navegação ganha uma sombra sutil. Automático
Animação reveal Elementos com a classe .reveal entram animados (fade + subida) quando aparecem na tela durante o scroll. Adicione a classe reveal a qualquer elemento HTML para que ele anime ao aparecer. Use reveal--delay-1, reveal--delay-2, etc. para atrasar a entrada.

Como adicionar animação reveal em novos elementos

No PHP/HTML, adicione a classe ao elemento:

<!-- Anima imediatamente ao entrar na tela -->
<div class="meu-elemento reveal">...</div>

<!-- Com atraso de 200ms (entra depois) -->
<div class="meu-elemento reveal reveal--delay-2">...</div>
